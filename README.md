# 🎮 3JS Game Developer Portfolio

An interactive 3D portfolio built with Three.js, featuring weather effects, post-processing magic, and AR support.

## ✨ Features

- 🌦️ **Dynamic Weather System** - Rain, Snow, Ash, and Floating Embers
- ✨ **Post-Processing Effects** - Bloom, Afterimage (Ghost Trail), Vignette + Chromatic Aberration
- 🎮 **Clickable 3D Objects** - Interactive portfolio items with custom color palettes
- 📱 **AR Support** - View your 3D model in Augmented Reality with Hiro marker
- 🎵 **Ambient Audio** - Background music with toggle
- 🌙 **Day/Night Cycle** - Dynamic lighting changes
- 🖱️ **Smooth Interactions** - Hover effects, tooltips, camera focus

## 🛠️ Technologies Used

- Three.js
- Vite
- WebXR / AR.js
- GLTF Models
- Custom Shaders

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/sirin-gh/3js-Project.git

# Navigate to project
cd 3js-Project/gothic-portfolio-2

# Install dependencies
npm install

# Run development server
npm run dev
